# Swedish English Teachers Classroom Adaptation Survey Dashboard

A comprehensive analytics tool designed to visualize and understand how Swedish English teachers adapt their lessons for students needing support and challenge.

üîó **Live Site:** [https://connectslide121.github.io/swedish-english-teac/](https://connectslide121.github.io/swedish-english-teac/)

## üìã Overview

This dashboard provides an interactive interface for analyzing survey data from Swedish English teachers. It focuses on two main aspects of classroom adaptation:
1.  **Support:** Strategies used to help students who are struggling.
2.  **Challenge:** Strategies used to challenge students who are advanced.

The tool allows researchers and stakeholders to explore the data through various lenses, including teacher experience, school type, and class size.

## ‚ú® Features

-   **CSV Data Upload:** Easily import survey results via CSV file.
-   **Interactive Dashboard:** Real-time updates and visualizations based on filtered data.
-   **Advanced Filtering:** Filter responses by:
    -   Teaching experience
    -   School type
    -   Certification status
    -   Class size
    -   And more...
-   **Rich Visualizations:**
    -   **Overview:** High-level summary statistics.
    -   **Factor Analysis:** Deep dives into Support and Challenge adaptation indices.
    -   **Group Comparisons:** Compare different teacher demographics.
    -   **Playground:** Create custom charts by selecting specific questions and grouping variables.
-   **Responsive Design:** Built with a modern, responsive UI using Tailwind CSS.

## üöÄ Local Development

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Connectslide121/swedish-english-teac.git
    cd swedish-english-teac
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:5173/` (or the port shown in your terminal).
